## kb-create-ts

### Initial version:

- pnpm required.
- uses:
  **Install globally**

```
pnpm i kb-create-ts -g
//or
npm i kb-create-ts -g
```
  **Execute with/out installing** 
```
pnpx kb-create-ts
//or
npx kb-create-ts
```
